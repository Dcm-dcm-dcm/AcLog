# AcLog
**AcLog, a log-based anomaly detection approach that incorporates human knowledge based on the framework of deep active learning.**
## Requirements
- python==3.6.13
- torch==1.10.2
- python_Levenshtein==0.20.7
- pandas==1.1.5
- tqdm==4.63.0

# Training
To begin training on HDFS, run this code:
```
python run.py
```
